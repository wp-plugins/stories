Sets up a stories post type. 

== Description ==

Currently this is just a "stories" post type. Moving forward, there will be story submission area.


== Installation ==

Simple install.

1. Upload the `stories` zip to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Can I change the slug? =

Yes! Just go to the settings page and change the slug or labels. Make sure to refresh your permalinks right after. This will flush the rewrites that control the slug.
